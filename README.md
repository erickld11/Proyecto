# EcoTrack Pro v2 🌱

Aplicación web para monitorizar consumos y huella de carbono.

## Tecnologías
- HTML
- CSS
- JavaScript
- Python FastAPI
- Docker
- Nginx

## Despliegue

### Clonar repositorio

```bash
git clone https://github.com/erickld11/Proyecto.git
```

### Ejecutar

```bash
cd ecotrack 
docker compose down -v
docker-compose up --build
```

### Acceso

Frontend:
http://localhost

Backend:
http://localhost:8000/docs

DB1:
http://localhost:8000/redocs
